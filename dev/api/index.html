API Reference

Tokens and request options

AzureIdentity.AbstractAzureCredential — Type

Abstract supertype for all Azure credentials. Concrete subtypes implement get_token (and optionally get_token_info, authenticate).

AzureIdentity.AzureAccessToken — Type

An access token returned by Azure AD: bearer string and absolute expiration timestamp.

AzureIdentity.AzureAccessTokenInfo — Type

Extended access-token information: bearer string, expiration, refresh-on hint, claims challenge, and tenant id.

AzureIdentity.TokenRequestOptions — Type

Per-call options passed to get_token/get_token_info: tenant id, claims challenge, and proof-of-possession parameters.

AzureIdentity.AuthenticationRecord — Type

Persistent record describing an authenticated user — username, home-account id, tenant id, authority, and client id. Returned by authenticate and reusable across processes via serialize_authentication_record / load_authentication_record.

AzureIdentity.TokenCachePersistenceOptions — Type

Options controlling on-disk persistence of the MSAL token cache (filename, allow-unencrypted-storage flag).

AzureIdentity.is_expired — Function

Return true if the access token has already expired (with a small safety margin).

Token acquisition

AzureIdentity.get_token — Function

Acquire an Azure AD access token for the requested scope(s) using the supplied credential. Returns an AzureAccessToken.

AzureIdentity.get_token_info — Function

Like get_token, but returns an AzureAccessTokenInfo with extended metadata (refresh-on hint, tenant id, claims).

AzureIdentity.authenticate — Function

Trigger an interactive sign-in flow (where supported) and return an AuthenticationRecord for cache reuse.

AzureIdentity.get_token_async — Function

Asynchronous wrapper around get_token, returning a Task{AzureAccessToken}.

AzureIdentity.get_token_info_async — Function

Asynchronous wrapper around get_token_info, returning a Task{AzureAccessTokenInfo}.

AzureIdentity.authenticate_async — Function

Asynchronous wrapper around authenticate, returning a Task{AuthenticationRecord}.

AzureIdentity.get_bearer_token_provider — Function

Return a zero-argument closure () -> bearer_string that lazily acquires (and refreshes) tokens for a given scope. Useful as a callback for HTTP clients that accept a token-provider lambda.

Authentication-record persistence

AzureIdentity.serialize_authentication_record — Function

Serialize an AuthenticationRecord to a JSON string.

AzureIdentity.deserialize_authentication_record — Function

Parse an AuthenticationRecord from a JSON string previously produced by serialize_authentication_record.

AzureIdentity.save_authentication_record — Function

Persist an AuthenticationRecord to a file path.

AzureIdentity.load_authentication_record — Function

Load an AuthenticationRecord from a file path previously written by save_authentication_record.

Service-principal credentials

AzureIdentity.EnvironmentCredential — Type

Credential that reads its configuration from AZURE_* environment variables (client id, secret/certificate, tenant id, etc.). Useful for production and CI environments.

AzureIdentity.ClientSecretCredential — Type

Credential that authenticates a service principal using a tenant id, client id, and client secret.

AzureIdentity.ClientAssertionCredential — Type

Credential that authenticates a service principal using a callback-supplied client assertion (typically a federated JWT).

AzureIdentity.CertificateCredential — Type

Credential that authenticates a service principal using an X.509 certificate (PEM/PFX file or in-memory bytes).

AzureIdentity.UsernamePasswordCredential — Type

Credential that authenticates a user with username and password using the resource owner password credentials (ROPC) flow. Not recommended for production; use only when other flows are unavailable.

AzureIdentity.WorkloadIdentityCredential — Type

Credential that authenticates a workload (e.g. AKS pod) via a federated identity token mounted by the Kubernetes service-account-token projection.

AzureIdentity.AzurePipelinesCredential — Type

Credential that authenticates an Azure DevOps pipeline service connection using a federated OIDC token.

AzureIdentity.OnBehalfOfCredential — Type

Credential that authenticates on behalf of an upstream user-assertion JWT — the OAuth 2.0 OBO flow used by middle-tier APIs.

Managed-identity credentials

AzureIdentity.ManagedIdentityCredential — Type

Credential that authenticates using an Azure Managed Identity (system-assigned or user-assigned). Probes the IMDS, App Service, Azure Arc, Cloud Shell, and Service Fabric endpoints automatically.

Chained / default credentials

AzureIdentity.ChainedTokenCredential — Type

Credential that tries a sequence of underlying credentials in order, returning the first token acquired successfully.

AzureIdentity.DefaultAzureCredential — Type

Convenience credential that probes (in order) EnvironmentCredential, WorkloadIdentityCredential, ManagedIdentityCredential, SharedTokenCacheCredential, AzureCliCredential, AzurePowerShellCredential, AzureDeveloperCliCredential, and (optionally) InteractiveBrowserCredential. The recommended starting point for most apps.

AzureIdentity.CachedCredential — Type

Wrapper that caches the access token returned by an inner credential until it (nearly) expires, avoiding redundant network calls.

AzureIdentity.clear_cache! — Function

Clear any access token cached by a CachedCredential (or compatible credential).

Developer credentials

AzureIdentity.AzureCliCredential — Type

Credential that requests a token by shelling out to the Azure CLI (az account get-access-token). Requires az login.

AzureIdentity.AzureCLICredential — Type

Alias for AzureCliCredential matching upstream azure-identity casing (AzureCLICredential).

AzureIdentity.AzurePowerShellCredential — Type

Credential that requests a token by shelling out to Azure PowerShell (Get-AzAccessToken). Requires Connect-AzAccount.

AzureIdentity.AzureDeveloperCliCredential — Type

Credential that requests a token by shelling out to the Azure Developer CLI (azd auth token). Requires azd auth login.

AzureIdentity.SharedTokenCacheCredential — Type

Credential backed by the MSAL shared token cache populated by other Microsoft developer tools (Visual Studio, etc.).

AzureIdentity.VisualStudioCodeCredential — Type

Credential backed by a token cached by the Visual Studio Code Azure Account extension.

Interactive credentials

AzureIdentity.DeviceCodeCredential — Type

Credential that prompts the user to complete the OAuth device-code flow at https://microsoft.com/devicelogin.

AzureIdentity.InteractiveBrowserCredential — Type

Credential that opens the system browser to complete the OAuth interactive authorization-code flow with PKCE.

AzureIdentity.AuthorizationCodeCredential — Type

Credential that exchanges a previously obtained authorization code for an access token (server-side flow).

Constants & utilities

AzureIdentity.AzureAuthorityHosts — Constant

Module containing well-known Azure AD authority hosts (AZURE_PUBLIC_CLOUD, AZURE_GOVERNMENT, AZURE_CHINA, etc.).

AzureIdentity.KnownAuthorities — Constant

Alias for AzureAuthorityHosts matching upstream naming.

AzureIdentity.DEVELOPER_SIGN_ON_CLIENT_ID — Constant

The 04b07795-… Microsoft developer-tooling client id used by Azure CLI / PowerShell / azd.

AzureIdentity.AZURE_OPENAI_SCOPE — Constant

Default OAuth scope for Azure OpenAI Cognitive Services (https://cognitiveservices.azure.com/.default).

AzureIdentity.get_openai_token — Function

Convenience wrapper that returns a bearer token suitable for Azure OpenAI calls — equivalent to get_token(cred, AZURE_OPENAI_SCOPE).token.

Exceptions

AzureIdentity.AbstractAzureAuthError — Type

Abstract supertype for all AzureIdentity errors.

AzureIdentity.AzureAuthError — Type

Generic Azure authentication error. Raised when no more specific subclass applies.

AzureIdentity.ClientAuthenticationError — Type

Raised when authentication failed and was rejected by the identity provider (e.g. invalid secret, MFA challenge unmet).

AzureIdentity.CredentialUnavailableError — Type

Raised when a credential is unable to attempt authentication (e.g. required environment variable missing). Caught by ChainedTokenCredential to fall through to the next credential.

AzureIdentity.AuthenticationRequiredError — Type

Raised when interactive authentication is required but cannot be initiated (e.g. headless environment with no fallback).

AzureIdentity.TokenCachePersistenceError — Type

Raised when the on-disk token cache cannot be read or written.
